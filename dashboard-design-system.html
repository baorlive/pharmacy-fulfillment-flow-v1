Typography

Only five type sizes, two font families, three line-height tokens, and normal tracking are allowed in dashboard UI.

Requires original prescription on delivery. --s-size-md / 14pxSpec: 400-700 / 1.2 or 1.5 leading / 0 trackingUse only for: default UI, body copy, buttons, tabs, badges
RX / PAID --s-size-sm / 12pxSpec: 500 / inherit leading / 0 trackingUse only for: micro labels, tiny counts, compact metadata
Family rule Use --s-font-sans for UI copy. Use --s-font-numeric only for order IDs, prices, counts, timestamps, and scan codes.
Token rule Use only --s-size-*, --s-leading-none, --s-leading-normal, --s-leading-body, and --s-tracking-normal. Do not use raw px, custom font stacks, or --f-* tokens in component CSS.
